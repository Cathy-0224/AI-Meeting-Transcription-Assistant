# AI Meeting Transcription Assistant (AI 會議轉錄助理)

> **工研院 (ITRI) 產業學院 — 實務實習專案 (2025.07 - 2025.08)**
> **由本人獨立完成從需求分析、系統設計到最終部署的完整開發流程。**

這是一款專為提升會議效率而設計的 AI 工具，整合了語音辨識與生成式 AI 技術，能夠在多種會議環境下提供即時的轉錄服務與 AI 摘要建議。

---

## 💡 核心功能 (Key Features)

* **即時語音轉錄 (Real-time Transcription)**：整合 Speech-to-Text API，實現高精確度的會議內容即時轉文字功能。
* **多環境適應性設定**：系統可針對不同的物理環境（如辦公室、會議室）進行設定，優化不同背景雜音下的辨識品質。
* **AI 即時回饋與記錄**：利用 Azure AI 服務對轉錄內容進行分析，自動生成會議紀錄要點、行動建議或即時回饋。
* **KM 知識管理整合**：有效將會議產出的非結構化語音資料轉換為結構化紀錄，幫助企業進行後續的知識管理與趨勢研判。

---

## 🛠️ 技術堆疊 (Tech Stack)

* **程式語言**: Python
* **雲端與 AI 服務**: 
    * Microsoft Azure AI Services (LLM 應用)
    * Speech-to-Text API (語音辨識整合)
* **開發工具**: Git, VS Code

---

## 🏗️ 系統流程說明 (System Workflow)

1. **設定參數** : 針對每次不同使用場景做不同設定。
2. **音訊擷取**：擷取會議現場音訊流。
3. **語音轉譯**：透過 STT API 將音訊即時轉換為文字數據。
4. **AI 分析**：將文字數據傳送至 Azure AI 模型進行內容理解。
5. **結果產出**：於介面即時呈現轉錄內容、會議摘要與建議事項。
